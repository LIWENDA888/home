关于自在造字

自在造字 (Zizao Type Studio) 致力于探索汉字在 可变字体 (Variable Font) 技术下的无限可能。我们认为设计应摆脱重叠参数的束缚，“可变，才自在”。

"Type design is not about drawing shapes, it's about programming a system."
— Zizao Type Studio Manifesto

我们构建的不止是静态的字形，而是一套可实时生长的视觉系统。在自在造字，每一个锚点的移动、每一条曲线的弹性，都通过严格的网格逻辑进行推演。

算法驱动的设计

我们将字体视为精密的几何算法，通过代码与设计的融合，实现从 Thin 到 Black，从 Condensed 到 Extended 的无级调节。

设计哲学

Geometric 几何构建

通过数学化的网格体系确立字形骨架，确保在任何极值下字形比例的稳定性。

Interpolation 动态插值

利用最前沿的插值技术，实现多轴向（字重、字宽、倾斜）的无缝切换。

常见问题

Q 购买授权后可以为多个客户做设计吗？

授权属于设计师本人或设计公司。您可以为无限量的客户提供设计作品（输出为图片或转曲后的矢量文件）。但如果您的客户需要直接获得字体文件进行二次编辑，则客户需要单独购买授权。

Q 可以在多台电脑上同时安装吗？

标准商业授权 (Standard Desktop) 允许在同一家公司内部的最多 5 台物理设备上安装。如果需要在更多设备上部署，请联系我们升级为企业版授权。

Q 支持发票开具吗？

支持。企业购买满 ¥200 可申请开具增值税普通电子发票。请在购买后将订单号及开票信息发送至财务邮箱。
